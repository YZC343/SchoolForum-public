# SchoolForum-public
# 前端运行说明：
安装Node.js,npm包管理器<br>
在该项目目录下打开命令行输入:<br>
cd front-end<br>
cd front-end<br>
npm install<br>
npm dev<br>
点击命令行中的url跳转到浏览器<br>
# 后端运行说明：
环境要求：<br>
python3.12<br>
flask<br>
flask_cors<br>
flask_sqlalchemy<br>
<br>
安装postgresql数据库<br>
在postgresql新建数据库schoolforum<br>
修改backend目录下的__init__.py文件中数据库配置中的用户名和密码<br>
运行__init__.py文件<br>
